Bot Command Manual

A comprehensive list of slash commands available in the Fenrix security bot, grouped by access levels.

Public Commands

These commands can be run by any member in the server.

Command Parameters Description
/help None Shows a helpful commands guide.
/infobot None Displays developer, support server, and general bot information.
/infoserver None Shows detailed counts and date of creation of the current server.
/infouser member (optional) Shows profile info, user ID, server join date, and roles of the target member.
/inforoblox username * Fetches Roblox API details (ID, display name, description, age) of the target player.
/vote None Displays top.gg vote portal buttons in English.
/invite None Provides the Discord OAuth2 link to invite the bot.
/discord None Sends the official Support Server invite link.

Staff & Moderation Commands

These commands require the user to have roles configured in the dashboard or Administrator permissions.

Command Parameters Description
/ban user *, reason (optional) Bans a user from the server and logs details in the logs channel.
/kick member *, reason (optional) Kicks a member from the server.
/unban user_id *, reason (optional) Unbans a user by their numerical ID.
/timeout member *, duration *, reason (optional) Restricts communication for a member (ranges from 5m up to 7 days).
/untimeout member *, reason (optional) Lifts a timeout from a member immediately.
/poll question *, opt1 *, opt2 *, opt3-10 (opt) Creates a multi-option voting poll with buttons and percentage bar visualizations.
/giveaway prize *, duration_seconds *, winners (opt) Starts an interactive giveaway. Draws winners and DMs them when time expires.
/partnership ping_role (opt), ping_everyone_here (opt) Triggers a popup text modal supporting paragraph lines and announcements.

Owner & Admin Commands

These commands require the user to have Administrator permissions on the server.

Command Parameters Description
/enableantispam None Enables the link protection filter for all channels.
/disableantispam None Disables the link protection filter.
/setup verify log_channel, roles_to_add, roles_to_remove, banner_url Posts a verification portal message with interactive buttons in the channel.
/setup welcome channel, text, banner_url Sets welcome message channel, description, and top banner.
/setup leave channel, text, banner_url Sets farewell leave channel, text, and banner.
/setup staff roles * Configures authorized role IDs for using staff/moderation commands.
/statusbot None Displays dynamic real-time bot statistics and generates a 24-hour activity graph.
* Parameters marked with * are required.
