📡 Channel System Logs

Configure where each category of system event gets logged. Granular control over every event type, with separate channels for each module.

⚖️

Moderation Logs

Logs all mod actions: /ban, /kick, /timeout, /warn, /mute, /lockdown. Includes moderator ID, target, reason, and timestamp.

🚪

Member Join / Leave Logs

Tracks every member join and leave event. Logs: User tag, ID, account age, and join time.

✏️

Message Edit / Delete Logs

Captures all edited and deleted messages. Shows original content, edit content, channel, and author.

🏷️

Role Change Logs

Logs role creations, deletions, and permission updates. Essential for detecting nuke attempts and unauthorized role manipulation.

📢

Channel Change Logs

Tracks channel create, delete, rename, and permission overwrites. Critical for monitoring unauthorized server modifications.

🔊

Voice Activity Logs

Logs voice joins, leaves, mutes, deafens, and channel moves. Useful for moderation transparency.

🔗

Invite Tracker

Shows which invite link each new member used to join. Helps identify the source of raids or spam bot waves.
