Privacy Policy

Last updated: June 2026

1. Introduction

This Privacy Policy explains how Fenrix collects, uses, and protects information when you use the Fenrix Discord bot or its web dashboard. We are committed to your privacy and handle all data responsibly.

2. Information We Collect

From Discord OAuth2 Login

When you log in via "Login with Discord", we receive:

  • Your Discord user ID, username, and avatar hash.
  • The list of servers you manage (used to show your server list on the dashboard).

This data is stored only in your browser session and is cleared when you log out.

From Guild Configuration

When a server administrator configures Fenrix via the dashboard, we store:

  • Channel IDs (for verification, logs, tickets, welcome, etc.)
  • Role IDs (for verification roles, staff roles, auto-roles, etc.)
  • Feature toggle states (anti-nuke, anti-raid, etc.)
  • Custom text fields (welcome messages, ticket descriptions).

We do not store message content, user DMs, or any personal communications.

Activity Data

Fenrix logs event counts per hour (e.g., "3 moderation events at 14:00"). This contains no user content and is purged after 48 hours.

3. How We Use Your Data

  • To operate the Fenrix dashboard and authenticate your session.
  • To apply your server configuration settings to the bot in real-time.
  • To display activity statistics on the Overview page.

We do not sell, share, or rent your data to third parties.

4. Data Storage & Security

Configuration data is stored in a secure database (Firebase Firestore or local JSON). Session data is encrypted using a server-side secret key. We use HTTPS for all web traffic on the production dashboard.

5. Data Deletion

To delete all data associated with your server, kick Fenrix from your server and open a ticket in the Support Server. All configuration data will be removed within 7 days.

6. Third-Party Services

Fenrix uses the following third-party services:

  • Discord API — Authentication, guild data, and bot functionality.
  • Google Firebase — Persistent database storage (optional).
  • QuickChart.io — Chart image generation (no personal data is sent).

7. Contact

For privacy concerns, join the Fenrix Support Server and open a ticket.
