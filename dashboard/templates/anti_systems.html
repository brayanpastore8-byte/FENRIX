🛡️ Anti-Systems

Advanced automated protection layers. All systems run independently — configure thresholds, durations, and actions per module.

📣

Anti-Mass Ping

Detects when a user mentions multiple members in a single message. Bots and admins are exempt.

🤖

Anti-Bot Influx (Join Gate)

If too many accounts join in a short time, the server enters lockdown mode. New joins require admin approval until the wave passes.

🔍

Anti-Scam Image Detection (OCR)

Uses OCR to scan image attachments for scam/phishing content (fake Nitro, prize giveaways, etc.) and auto-deletes them.

OCR scanning requires pytesseract to be installed on the server. If unavailable, this module will gracefully disable itself without breaking other features.

👻

Anti-Ghost Ping Detection

Detects when a user pings someone and immediately deletes the message to hide it. Logs the ghost ping with the original content.

🔐

Lockdown Configuration

Defines which channels are locked during a /lockdown command or automatic security lockdown event.

Leave empty to lock ALL text channels during a lockdown.

🔐

Anti-File-Hack

Blocks dangerous file uploads (.exe, .bat, .cmd, .ps1, .vbs, .dll, .scr, .msi, .jar, .py, .sh, etc.) that can be used to hack the server via CMD/PowerShell/RCE. Admins and staff are exempt.
